使用&自编说明：
格式 
    {
        “性别”：{
            “项目”：{
                “年级”：[{
                    “100分”：100分对应的标准
                }, “计算类型”]
            }
        }
    }
{

“计算类型”分为：
    （1）“num”：比对标准为纯数字，可有小数，数越大分越高
    （2）“time-short”：比对标准为纯数字，时间越短越高
    （3）“time-long”：比对标准为x'y"：x分y秒，时间越短分越高
